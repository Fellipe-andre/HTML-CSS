Estrutura HTML 5

Corpo da Estrutura

<!DOCTYPE html > - Indica o tipo de documento (html)

< html lang="pt-br" > - Indica o começo do site e o idioma do site(serve mais para nao usar o google tradutor)

< head > - Indica que será o começo da cabeça do documento (as configurações ficam aqui)

< meta charset="UTF-8"> - Indica que tipo de caracteres irá ser usado no arquivo(UTF-8 é brasileiro)

< meta http-equiv="X-UA-Compatible" content="IE=edge"> - (Descobrir o significado)

< meta name="viewport" content="width=device-width, initial-scale=1.0"> - (Diz que vai usar a pagina inteira do navegador

< title > < /title > - Título do HTML que ficará na aba do navegador

< /head > - Indica que será o dim da cabeça do documento (as configurações terminam aqui)

< body > - Começo do corpo (tudo que aparecerá no site ficará aqui)

< /body > - Fim do corpo

< /html > - Fim do site

Irarquia de títulos

  • < h1 > < /h1 > - 1° na hierarquia

  • < h2 > < /h2 > - 1° na hierarquia

  • < h3 > < /h3 > - 1° na hierarquia

Páragrafos

    < p > < /p > - para inserir um Páragrafo

LINKS e TAGS ÚTEIS

    < abbr > < /abbr > - para abreviar algo

  • exemplo:

    posicione o mouse aqui

    • < a href="pagina 2">clique aqui para ir para a pagina 2 < /a > - para criar um link para outro lugar (seja pagina 2 ou arquivos externos)

  • exemplo:

    clique aqui para ir para a pagina 2
